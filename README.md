# 🚚 Coal Transportation Monitoring Dashboard

## 📌 Overview
The Coal Transportation Monitoring Dashboard is a real-time web-based system designed to track and manage coal logistics efficiently. It integrates IoT-enabled GPS devices with a dynamic dashboard to provide live updates on vehicle location, movement, and operational status.

---

## 🎯 Objectives
- Enable real-time tracking of coal transportation vehicles  
- Improve logistics efficiency and transparency  
- Reduce delays and unauthorized route deviations  
- Provide actionable insights through live data visualization  

---

## ⚙️ Tech Stack
- **Backend:** Node.js  
- **Frontend:** HTML, CSS, JavaScript  
- **APIs:** REST APIs  
- **Hardware:** GPS Module (IoT Device)  
- **Data Handling:** JSON / API Integration  

---

## 🚀 Features
- 🔹 Real-time vehicle tracking using GPS data  
- 🔹 Interactive dashboard with live updates  
- 🔹 Route monitoring and tracking  
- 🔹 API-based data communication  
- 🔹 User-friendly interface for logistics management  

---

## 🧠 System Architecture
1. GPS-enabled IoT devices collect location data from vehicles  
2. Data is transmitted to the backend server via APIs  
3. Node.js processes and manages incoming data  
4. Dashboard displays real-time information using frontend technologies  

---

## 🔄 Workflow
1. Vehicle equipped with GPS module starts journey  
2. Location data is continuously captured  
3. Data is sent to server through REST API  
4. Server processes and stores the data  
5. Dashboard updates in real-time for users  

---

## 📊 Output / Results
- Accurate real-time tracking of transportation vehicles  
- Improved visibility of logistics operations  
- Reduced manual monitoring effort  
- Enhanced decision-making through live data
